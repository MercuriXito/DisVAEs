Implementation Plans:

+ Models:
    - [ ] VAE
    - [ ] BetaVAE
    - [ ] AnnealVAE
    - [ ] DIP-VAE
    - [ ] beta-TCVAE
    - [ ] FactorVAE
+ Metrics:
    - [x] BetaVAE Scores
    - [ ] FactorVAE Scores
    - [ ] MIG
    - [ ] SAP

Currently, only `dsprite` dataset supported. More training results would be released 
currently.
